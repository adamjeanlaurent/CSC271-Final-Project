Code Implementation of Bubble Sort

Introduction

Despite bubble sort being one of the slowest sorting algorithms, it is one of the easiest to implement from scratch. It can be implemented in several programming languages. In many program languages, bubble sort can be either implemented iteratively(loops) or recursively. For now, all the implementations of bubble sort we will show you will be implemented iteratively. However, you are more than welcome to search for recursive solutions in other places. Again, the following implementations will not use the most optimal solution.
