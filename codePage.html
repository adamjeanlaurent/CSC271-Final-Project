Code Implementation of Bubble Sort

Introduction

Despite bubble sort being one of the slowest sorting algorithms, it is one of the easiest to implement from scratch. It can be implemented in several programming languages. In many program languages, bubble sort can be either implemented iteratively(loops) or recursively. For now, the implementation of bubble sort will be implemented iteratively. However, you are more than welcome search for for recursive solutions in other places. Again, the following implementations will not use the most optimal solutions.

# Python program for implementation of Bubble Sort

def bubbleSort(arr):
        n = len(arr)

        # Traverse through all array elements
        for i in range(n):

                # Last i elements are already in place
                for j in range(0, n-i-1):

                # traverse the array from 0 to n-i-1
                # Swap if the element found is greater
                # than the next element
                if arr[j] > arr[j+1] :
                                arr[j], arr[j+1] = arr[j+1], arr[j]

# Driver code to test above
arr = [64, 34, 25, 12, 22, 11, 90]
bubbleSort(arr)

print ("Sorted array is:")
for i in range(len(arr)):
        print ("%d" %arr[i]), end=" ")
